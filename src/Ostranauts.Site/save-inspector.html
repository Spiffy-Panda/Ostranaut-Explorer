Upload an Ostranauts save zip (the <name>_<epoch>.zip file from your Saves/<name>_<epoch>/ folder) and pick a ship inside it to inspect. The save-format per-ship JSON is a runtime superset of data/ships/*, so the same component checklist, floor plan, and room-classification cards apply. Component counts reflect the live build state at the moment the save was written, including damaged/uninstalled items.
